# PIPBench — Brand Guidelines (v1)

## 1. Who we are
PIPBench is the owner-side PIP benchmark. We turn a brand-mandated renovation plan from a bill into a negotiation. The brand personality is a **surveyor, not a guru**: precise, sourced, calm, on the owner's side of the table. We never bluster, never overclaim, and we show our work — every benchmark carries a named public source and an honest confidence grade.

**Tagline:** *Your PIP isn't a bill. It's an opening offer.*
**Functional descriptor:** The owner-side PIP benchmark.

## 2. Logo system
Files in `run-1/brand/`:
- `logo.svg` — primary horizontal lockup, light backgrounds
- `logo-dark.svg` — lockup for dark backgrounds
- `mark.svg` — standalone mark (navy tile), avatars/app icons
- `favicon.svg` — 32px simplified favicon
- `logo-candidates/` — concepts A–D + contact sheet (A chosen; rationale in critique below)

**The mark** is a surveyor's benchmark: an amber datum bar above a broad arrow. A surveyor's benchmark is a physical, verified reference point carved where everyone can check it — exactly what the product is. The arrow rises *to* the datum: the owner bringing the bid up against the reference, not the reference descending on the owner.

**Selection rationale (critique loop record):** four concepts were generated and reviewed at full size and 40px. A (benchmark) won on distinctiveness + meaning + small-size legibility. D (PB monogram) was solid but generic; C (bars) read as generic analytics; B (range band) had the best story but muddy small-size execution. Contact sheet: `logo-candidates/contact-sheet.html`.

**Usage rules:** clear space = height of the datum bar on all sides; never recolor the arrow outside brand inks; never place the light lockup on photos without a paper-tone panel; the amber datum bar never appears without the arrow (it would read as a warning stripe).

## 3. Color
| Role | Name | Hex |
|---|---|---|
| Primary ink | Datum Navy | `#10293D` |
| Secondary ink | Slate | `#3D5A73` |
| Background | Survey Paper | `#F7F4EE` |
| Accent / brand | Datum Amber | `#E8A317` |
| Semantic: in range / verified | Range Green | `#2F7D4F` |
| Semantic: above range / flag | Flag Red | `#BF4034` |
| Muted text | Field Gray | `#6B7A88` |

Semantics are load-bearing: green = *in benchmark range* or *Verified source grade*; amber = *questioned / directional*; red = *above range / flagged*. Never use these three decoratively.

## 4. Typography
- **UI & headlines:** Segoe UI stack (`"Segoe UI", -apple-system, "Helvetica Neue", Arial, sans-serif`), bold (700) headlines with tight (-1%) letter-spacing.
- **The memo:** Georgia stack (`Georgia, "Times New Roman", serif`) — the negotiation memo is a formal business document and should look like one the moment it renders.
- **Evidence & figures:** `Consolas, "Courier New", monospace` for citations, benchmark figures, and confidence badges — evidence reads like the record, with tabular alignment.
- Rationale: the UI persuades, the memo argues, the evidence testifies. Three registers, three faces. (Launch-plan note: swap Segoe stack for a licensed webfont — e.g., Inter — and vectorize the wordmark before public use.)

## 5. Voice
- **Owner-side, not anti-brand.** We help owners negotiate, we don't bash franchisors. The brand relationship survives the negotiation.
- **Sourced or silent.** Every number in marketing carries a source that actually exists. If we can't source it, we don't say it. [Estimated] figures are labeled, even in marketing.
- **Calm confidence, no guru-ism.** Competitors called themselves "guru" and "genius." We are named after a reference point. Sentences are short. Claims are checkable.
- Words we use: benchmark, datum, range, evidence, memo, sourced, verified, directional.
- Words we never use: guaranteed savings, insider, secret, hack, weapon; and never legal/engineering-advice framing ("your PIP is unlawful", "this spec is unsafe").

## 6. Product UI signatures
- **Confidence badges:** `VERIFIED` (green, primary-sourced benchmark) and `DIRECTIONAL` (amber, interpolated/escalated) — monospace, always visible next to every benchmark. This is the brand's honesty made visible; never hide or collapse it.
- **Range bars:** every category benchmark renders as a horizontal range band (low–avg–high ticks) with the bid's marker dot — green in range, red above, slate below.
- **Source lines:** every benchmark shows "Source: [name, year]" in monospace, linked in-product to the methodology page.
